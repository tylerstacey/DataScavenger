What is Data Scavenger?

  • Data Scavenger is a web application that allows users to collaborate and share data.

Main Features

  • Manage data for multiple projects.
  • Help others collect data for their projects.
  • Generate tables for your projects and view reports published by other's.
  • Users can create projects and add data to projects.
  • The administrator can delete users, projects, and data.
